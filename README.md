# Handwritten Digit Recognition using MNIST

This project implements a neural network to classify handwritten digits (0–9)
using the MNIST dataset.

## Objective
Build a model that can correctly identify handwritten digits from grayscale images.

## Model Details
- Input: 28×28 grayscale images
- Hidden Layer: Dense layer with ReLU activation
- Output Layer: Softmax (10 classes)

## Results
- Test Accuracy: **97.79%**

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
